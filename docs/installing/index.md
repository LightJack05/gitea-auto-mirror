# Installation
Gitea Auto Mirror is available as a docker container and can be deployed everywhere where docker containers run. Therefore, how you install Gitea Auto Mirror will depend on your platform.

## Docker Image
The docker image is available from GHCR [here](https://github.com/LightJack05/gitea-auto-mirror/pkgs/container/gitea-auto-mirror).

## Installation platforms
You can install Gitea Auto Mirror in [docker compose](docker-compose.md) and [Kubernetes](kubernetes.md) depending on what fits better into your infrastructure.
